# type

> 项目类型文件

## 目录说明

```
  |-- app.type.ts 产品、集群等类型
  |-- common.type.ts 通用类型：用户、分页、排序、搜索等
  |-- event.type.ts 事件模块类型
  |-- object.type.ts 对象模块类型
  |-- parameter.type.ts 参数模块类型
  |-- requirement.type.ts 需求模块类型
  |-- table.type.ts 表格相关类型
  |-- template.type.ts 参数模板类型
  |-- terminal.type.ts 终端管理类型
  |-- version.type.ts 终端、事件类型参数版本相关类型
```
