# components

> 项目通用组件

## 目录说明

```
  |-- ant-design
      |-- index.ts antd组件导入
  |-- common 通用组件
      |-- m-filter-select 支持虚拟滚动的下拉选择组件
      |-- u-divider.vue 分割线组件
      |-- u-filter-select.vue 筛选组件，常用于列表页筛选
      |-- u-single-select.vue 支持虚拟滚动的单选组件
      |-- u-tag.vue tag 组件
  |-- layout 布局组件
      |-- topbar topbar 组件
      |-- u-collapse.vue 折叠面板组件
      |-- u-info-item.vue 信息 item 组件
      |-- u-info.vue 信息组组件，子节点为 u-info-item
      |-- u-layout.vue 全局布局组件
      |-- u-left-right.vue 左右布局组件
      |-- u-sidebar.vue 侧边栏组件
      |-- u-topbar.vue 顶部栏组件
  |-- modal 弹窗组件
      |-- ImagePreviewModal.vue 图片预览弹窗
  |-- specific 业务组件
      |-- s-editor.vue 富文本组件
      |-- s-page-top.vue 页面顶部布局组件
      |-- s-table-search.vue 列表页搜索、刷新组件
      |-- s-upload-image.vue 图片上传组件（用于坑位创建、编辑）
      |-- s-user-select.vue 支持分页搜索的用户选择组件
      |-- upload-request.js 图片上传请求封装
  |-- register.ts 组件注册
```
